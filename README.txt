== Description ==

Wine Club Connection is an easy to use Woocommerce plugin to give your store recurring club functionality. Ideal for Wine Club Connection, subscription boxes, or any business based on recurring product memberships.

Key features:

Membership system creation
Membership discounts 
Shipping Discounts for Members
Use WooCommerce Products for Automated Member Enrollment
Restrict Products based on membership levels

Pro add on:

Batch & Automated Billing
Square POS 2-way 

Repeat purchases drive brand loyalty – a customer loyal to a brand becomes an advocate who will refer future customers month over month.

== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of Club Connection, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type “Wine Club Connection,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the Wine Club Connection plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

e.g.

1. Download "Wine Club Connection" plugin
2. Upload 'wine-club-connection' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
